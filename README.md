# creepypasta_lore

공식 문서 형식으로 작성된 크리피파스타 로어 아카이브입니다.

각 문서는 특정 기관, 시설, 혹은 체계의 내부 지침서 형태로 구성되어 있으며, 
기존 세계관과의 직접적 연결을 명시하지 않습니다.

독자는 문서의 형식과 반복되는 구조, 그리고 미묘한 불일치를 통해 
서서히 전체 그림을 추론하게 됩니다.

- 내부 문서 형식 유지

- 반복되는 문장 구조 활용
- 현실과 허구의 경계 모호화
